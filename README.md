# Wumpus

#### Symbols:
- Breeze: pit nearby
- Stench: Wumpus nearby

#### Keys:
- `Right/Left`: turn right/left
- `Up`: move forward
- `Space`: shoot (one shot only)
- `Enter`: grab (if gold present)

#### Rules: 
- see [Wumpus World](http://cis-linux1.temple.edu/~giorgio/cis587/readings/wumpus.shtml)
